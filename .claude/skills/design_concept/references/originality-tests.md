# Originality Tests

Apply these tests to concepts and mockups. Original means product-owned and functionally distinct, not merely unfamiliar.

## Blur Test

Blur the design until text disappears.

Reject it when:

- its silhouette matches a reference screen;
- all concepts reduce to the same sidebar, hero, and card grid;
- the visual hierarchy depends entirely on labels.

## Logo-Swap Test

Imagine replacing the product logo and accent color.

Reject it if the result could pass unchanged for an unrelated SaaS product. Strengthen domain-specific structure, content, and interaction.

## Reference-Distance Test

For every reference, separate:

- `preserve`: craft, density, typography, atmosphere, interaction quality;
- `do not copy`: layout, centerpiece, metaphor, silhouette, information placement.

Reject a concept that preserves the second list.

## Functional-Artifact Test

Ask what the dominant visual element does.

Reject it if it:

- encodes no information;
- cannot respond to product state;
- competes with the primary action;
- disappears without affecting usability;
- exists mainly to make the screenshot dramatic.

## Metaphor Load Test

The user should not need to learn a poem before using the product.

Reject metaphors that:

- require explanatory text;
- distort familiar actions;
- hide exact values that matter;
- create emotional mismatch with serious tasks.

## Derivative-Family Test

Compare candidates by:

- dominant geometry;
- focal point;
- reading path;
- interaction;
- information hierarchy.

If three or more match, they are one concept with different styling. Keep one and generate new structures.

## Product Reality Test

Verify that the design can represent:

- no data;
- typical data;
- dense data;
- stale or failed data;
- accessibility and reduced motion;
- the actual actions and data available.

Reject concepts that work only with perfect fixture data.

## Screenshot Trap Test

Ask whether the design would remain good after daily use.

Penalize:

- illegible microcopy;
- excessive glow;
- constant motion;
- hidden controls;
- decorative density;
- a centerpiece that leaves insufficient space for real content.
