# Concept Forge

## Divergence Matrix

Generate candidates by crossing choices from each row:

| Axis | Possibilities |
|---|---|
| Information model | state, sequence, territory, relationship, transformation, comparison, conversation, archive |
| Artifact | instrument, material object, field, editorial surface, spatial environment, kinetic system, adaptive typography |
| Spatial grammar | centered, edge-driven, asymmetric split, layered depth, continuous surface, progressive disclosure, panoramic |
| Interaction | reveal, tune, assemble, focus, scrub, hold, draw, navigate, unfold, transform |
| Emotional posture | calm, decisive, protective, reflective, energetic, clinical, playful, ceremonial |
| Time behavior | snapshot, accumulation, decay, phase shift, trace, forecast, convergence |

Generate at least ten candidates privately. Ensure the candidates differ structurally, not only aesthetically.

## Functional Artifact Contract

Define the artifact in one sentence:

> [Artifact] maps [real data] to [visible behavior] so the user can [primary action].

Reject it if:

- it encodes no data;
- the page works identically after removing it;
- it hides exact values that matter;
- it cannot represent empty, dense, and error states;
- it competes with the primary action;
- it exists mainly for a dramatic screenshot.

## Candidate Score

Score 1–10:

- immediate clarity;
- daily usefulness;
- product ownership;
- distance from sibling motifs;
- emotional fit;
- accessibility;
- responsive potential;
- technical feasibility;
- dense-state resilience;
- motion restraint.

Reject any candidate below 7 in clarity, usefulness, accessibility, or feasibility.

## Concept Stress Tests

### Product Reality

Represent actual content, long names, missing values, stale data, permissions, errors, and maximum expected density.

### Metaphor Load

The user must not need to learn a poem before using the product. Exact values and familiar actions remain explicit.

### Screenshot Trap

Penalize illegible microcopy, excessive glow, constant motion, hidden controls, decorative density, and centerpieces that consume working space.

### Reference Distance

Separate every reference into:

- craft to preserve;
- structure not to copy.

Reject any candidate that retains the reference silhouette, focal object, or information placement.

## Decision-Complete Concept

Specify:

1. first, second, and third read;
2. artifact meaning and state mapping;
3. primary action and signature interaction;
4. desktop composition;
5. mobile recomposition;
6. empty, dense, loading, stale, error, permission, and reduced-motion behavior;
7. preserved DNA;
8. avoided motif collisions;
9. implementation risk.

Select one concept. Hybrids usually dilute hierarchy and originality.
